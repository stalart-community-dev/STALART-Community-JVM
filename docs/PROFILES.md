# Профиль конфигурации

В ветке JDK 25 используется один основной профиль: `stable`.

## Что это значит

- основной файл конфигурации: `configs/stable.json`;
- активный профиль хранится в `HKCU\\Software\\StalartJvmWrapper`;
- `cli.exe --autotune` просто устанавливает активным `stable`;
- при `Reset Config` профиль `stable` пересоздаётся значениями по умолчанию.

## Когда нужны другие JSON-файлы

Кастомные профили можно добавлять вручную в `configs/`, затем переключать через `Select Config`.
Но production-базой остаётся `stable`.

## Где смотреть результат
